History

  • 1996 "frame-based" layout model
  • 2005 Advanced Layout Module
  • 2012 draft Grid Layout spec
  • 2017 major browsers support an implementation of CSS Grid
  • 95.45% of browsers are supported globally
  • If your app supports IE you can use autoprefixer
  • A grid is an intersecting set of horizontal and vertical lines
  • One set defines the columns and the other the rows

Grid has the following functions:

  • Fixed and flexible track sizes
  • Item placement
  • Creation of additional tracks to hold content
  • Alignment control
  • Control of overlapping content
